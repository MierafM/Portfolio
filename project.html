Song Reccomendation App

I single-handedly developed a React app that utilizes the Spotify Web API to provide users song recommendations based on their recently played songs.The app displays user’s recently played tracks and generates a curated set of recommendations for each song by analyzing each track’s metadata given from Spotify’s API. It also updates recently played songs in real-time as well as display a given track’s audio features and popularity.

languages & libraries used:

React, JS, HTML, CSS
